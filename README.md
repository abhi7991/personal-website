# 🚀 Personal Portfolio: Abhishek Shankar

A clean, minimalist single-page portfolio designed to showcase a journey from Chennai to Silicon Valley. Built with **pure HTML, CSS, and JavaScript**—zero frameworks, zero dependencies, and lightning-fast load times. Vibe coded and deployed on Vercel

---

## 🛠️ Tech Stack
* **Frontend:** Semantic HTML5, Modern CSS3 (Flexbox/Grid), Vanilla JavaScript.
* **Philosophy:** No build tools, no `node_modules`, just clean code that works in any browser.
